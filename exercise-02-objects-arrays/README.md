# Exercise 2: Objects and Arrays Basics

## Learning Objectives
- Create and manipulate objects
- Create and manipulate arrays
- Access object properties using dot and bracket notation
- Access array elements using indices
- Add, modify, and remove items from arrays and objects

## Instructions
Complete the tasks in `index.js`. Run the file with `node index.js` to test your code.

## Tasks
1. Create objects to represent course and assignment data
2. Create arrays to store multiple assignments and submissions
3. Access and modify object properties
4. Access and modify array elements
5. Add and remove items from arrays

## SBA Connection
This exercise teaches you how to:
- Structure data like CourseInfo, AssignmentGroup, and LearnerSubmission objects
- Store multiple assignments in arrays
- Access and manipulate the data structures you'll use in the SBA
