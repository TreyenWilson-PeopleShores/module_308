# Exercise 6: Functions

## Learning Objectives
- Create reusable functions
- Use parameters and return values
- Understand function scope
- Break down complex problems into smaller functions

## Pre-Exercise Quiz
Test your knowledge before starting:
1. Which keyword is used to declare a function in JavaScript?
2. How do you produce a value from a function in JavaScript?
3. What is the term for a function that takes another function as an argument?
4. How can you define an anonymous function in JavaScript?
5. How can you set a default parameter for a function in ES6 JavaScript?
6. What is the primary purpose of arrow functions in ES6?

## Instructions
Complete the tasks in `index.js`. Run the file with `node index.js` to test your code.

## Tasks
1. Create simple functions with parameters and return values
2. Create helper functions for common calculations
3. Call functions from within other functions
4. Build functions that will be useful for the SBA

## SBA Connection
This exercise teaches you how to:
- Create the getLearnerData() function required by the SBA
- Build helper functions for calculating percentages, averages, etc.
- Organize your code into reusable pieces
- Make your code more maintainable and testable
